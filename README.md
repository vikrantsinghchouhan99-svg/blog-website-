# blog-website-
📝 Blog Website

A responsive and modern Blog Website built using HTML, CSS, Bootstrap, and JavaScript. This project provides a clean user interface for displaying blog posts, categories, and featured content with a mobile-friendly design.

🚀 Features
Responsive design using Bootstrap
Modern and attractive UI
Blog post cards and layouts
Navigation bar with smooth user experience
Featured blog section
Mobile and desktop compatible
Clean and organized code structure
🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
JavaScript
